1️⃣ Register a new user

Method: POST
URL: http://localhost:8888/users/register

Body (JSON):

{
  "username": "newuser",
  "password": "mypassword"
}


2️⃣ Method: POST

URL: http://localhost:8888/users/login

Body (JSON):
{
  "username": "vidal",
  "password": "123"
}

3️⃣ Get user by ID (protected)

Method: GET
URL: http://localhost:8888/users/1

Headers Tab:
Key, Value
Authorization, 1

4️⃣ Method: GET
URL: http://localhost:8888/playlists

Headers tab:
Key,	Value
Authorization,	1

5️⃣ Create a new playlist

Method: POST
URL: http://localhost:8888/playlists

Headers Tab:
Key, Value
Authorization, 1

Body (JSON):
{
  "title": "My New Playlist"
}

6️⃣ Add a track to a playlist

Method: PUT
URL: http://localhost:8888/playlists/1

Headers Tab:
Key, Value
Authorization, 1

Body (JSON):
{
  "name": "Imagine",
  "artist": "John Lennon",
  "album": "Imagine",
  "mbid": "johnlennon|imagine",
  "image": "https://example.com/imagine.png"
}

7️⃣ Delete a playlist

Method: DELETE
URL: http://localhost:8888/playlists/1

Headers Tab:
Key, Value
Authorization, 1

8️⃣ Search for a track on Last.fm

Method: GET
URL: http://localhost:8888/tracks/search?track=Bohemian%20Rhapsody

Headers Tab:
Key, Value
Authorization, 1

With Fuzzy search:
URL: http://localhost:8888/tracks/search?track=bohemian%20rhapsody&fuzzy=true

9️⃣ Get detailed info for one track

Method: GET
URL: http://localhost:8888/tracks/36d11386-4f8c-444f-939e-b7abf8ebab0e

Headers Tab:
Key, Value
Authorization, 1

If no MBID can be found, the fallback URL
URL: http://localhost:8888/tracks/Coldplay|Yellow
